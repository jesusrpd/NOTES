#COMANDOS BÁSICOS

CHANGE DIRECTORY -> Cambiarnos de directorios (cd)

AL PONER "cd .." NOS REGRESAMOS A UN DIRECTORIO ANTERIOR

LIST -> ENLISTAR EL CONTENIDO QUE TIENE UNA CARPETA (dir)

EXIT -> CERRAMOS LA TERMINAL (exit)

CLEAR -> LIMPIAMOS LA CONSOLA (cls)

HELP -> MOSTRARNOS AYUDA

MAKE DIR -> CREA CARPETAS (md #nombre de la carpeta)

MOVE -> MOVER CARPETAS O ARCIVHIVOS A OTRO LADO (move #que_archivo_mover #a_donde_se_va_a_mover)
